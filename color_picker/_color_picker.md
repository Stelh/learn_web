# Features

- Roue de couleur hexa
- Jauge d'opacité
- Bouton "Profile" pour créer des profile
- Bouton "Add color" pour ajouter la couleur au profile
- Modification en live de l'hex color
- Possibilité d'introduire hex d'appliquer la couleur

## Résumé de l'outil

Pouvoir afficher sur une page différents profile. Ces profiles regrouperont différentes couleurs
